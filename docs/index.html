Sevenbridges

Introduction

Graphs are a natural way of representing many data sets and problem types. Sevenbridges is a Dojo module which makes it easy to display and interact with graphs within a web browser.

Sevenbridges requires browser support for SVG.

Graph Representation

Sevenbridges is built on top of dojo.data. Any data store supporting the Read, Write, Identity and Notification APIs may be used. Vertices and edges may be dynamically created, deleted and updated through the dojo.data API.

Sevenbridges interprets items in a data store as vertices or edges, based on the presence of a "type" field. Items with no type or an unrecognized type are ignored.

typemeaning
vertexitem is a vertex
edgeitem is an edge

In vertex items, the following fields are recognized. All other fields are ignored. We assume that the data store is configured such that the identifier field is "id" and the label field is "label".

fieldmeaning
ida unique identifier
labelvertex label (optional)
hrefhyperlink URI (optional)
weightweight of vertex (default 1)
classDOM class of vertex (optional)

In edge items, the following fields are recognized. All other fields are ignored. We assume that the data store is configured such that the identifier field is "id" and the label field is "label".

fieldmeaning
ida unique identifier
labeledge label (optional)
sourceidentifier of source vertex
targetidentifier of target vertex
weightweight of edge (default: 1)
classDOM class of edge (optional)

By way of example, a simple graph is defined in example.json in a format suitable for a dojo.data.ItemFileWriteStore.

Graph Layouts

Sevenbridges supports several graph layout strategies. Different layouts may be applied to a graph at any time.
WidgetFunction
sevenbridges.EllipticalLayout Layout vertices in an ellipse
sevenbridges.ForceDirectedLayout Layout vertices using a force directed simulation
sevenbridges.FastForceDirectedLayout A faster force directed layout
sevenbridges.RandomLayout Layout vertices randomly

The force directed layouts often provide esthetically pleasing graph layouts.

Graph Widget

A graph is displayed by the sevenbridges.Graph widget. This widget supports the following parameters:
ParameterSemantics
storeunderlying data store
layoutlayout to apply
directeddirected graph (default: false)
